filter 變色特效

五月 31, 2020 CSS, 屬性
簡介原圖只是要改變顏色不用再進到繪圖軟體修改圖，只要用 filter 的 CSS 屬性就可以，雖然不能作到細節修飾，但對只是要作復古、灰階的黑白圖之類的效果還是很方便。 另外在按鍵也可以使用，只需要設定全部按鍵的 :hover 、:active 、:focus 會改變的亮暗度、飽和度後，在個別按鍵設定顏色就會有一般或按下時不同的顏色，加上設定的是個別按鍵所以顏色不同但都可以有一致性的變化。
