FLEX 排版

六月 7, 2020 CSS

簡介

FLEX 的對齊方法，對齊排版的方法就像是開文件編輯器一樣，先設定範圍 (外層) 再放入要編排的文字、圖片 (內層的內容)。

HTML

在要使用 FLEX 的元素外一定要用一個 外層容器 包著全部的 內層 ，而以下的例子都是用上方的 HTML 直接在 CSS 設定作示範。

外層容器

把排版的範圍想成一張 A4 的紙，在外層加入 display:flex 就等於開啟一份有文字圖片排版的文件

讓內容的元素都可以套用到 FLEX 的方法作到像 Google 文件 的對齊方式

排列方向

排列方向就等於是文件的對齊方式，而要靠左、靠右的設定是用 flex-direction

轉成直式的 由上至下、由下至上

換行

預設在內容太多時，會自動縮小全部內容的寬度，讓全部內容都能被看到，但這樣就會改變原來的樣子了，當內容超過外層容器時要換行就可使用 flex-wrap

對齊方式

  • 水平 X 軸對齊 justify-content
  • 垂直 Y 軸對齊 分兩種
    • 當內容只有一行時 align-items
    • 內容有多行時 align-content

水平的左右對齊

垂直的上下對齊

上下對齊分 單行的 align-items 、多行的 align-items 屬性對齊方式，值除了多 space-around 、space-between 的不同，其他都是相同的。

要注意若是有設定換行是使用 flex-wrap: wrap-reverse 由下至上換行時，上下排列的方向就會是相反的由下至上排列。
