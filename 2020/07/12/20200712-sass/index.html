簡介

在編寫 CSS 時常會為了一個效果而寫一大長串的樣式、編排，又或者是使用了許多幾乎有著相同文字效果的元素，讓整個 CSS 檔案很龐大之外，在之後的修改維護也十分的不方便。

而使用 SASS 或 SCSS 就可以用巢狀的方式，一個包一個的整理整齊，兩者最大的不同是 SCSS 還保留著大括號 { } 和分號 ; ，SASS 就不需要了。

Variable 變數

定義變數的方式是使用 $ 加變數名稱，而且不能有空格。

@mixin 和 @include

這就像在使用 function bluebox () { } 的作法

打包用 @mixin 把全部的樣式都放進來

要使用時是用 @include 把包在裡面的置入

而且還可以加上變數配合計算產生使用

@import

最後還可以參考 Bootstrap 的方式把各類的樣式拆分多個 SASS / CSS 檔案後，再用 @import 加檔案名和副檔名 匯入，而副檔名也可省略。
