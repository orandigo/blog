reduce() 陣列加法

十月 18, 2020 JavaScript, 函式

前言

加法要注意陣列內的值是數字或是字串，這將會決定最後的值是 數值 或是 字串 。

以下列 3 個陣列為例計算

最後會得到一個數值 7 ，為累計 0 + 1 + (-1) + 2 + 5 得出的值

同樣加法的方式使用在 全部字串

會出現 0 是因為未計算前給了 sum 一個值，所以累計時也被加入了。

和 混用 時

會發現和前一個陣列計算結果只差第一個 0 ，這是因為一開始是以數字計算，直到出現值是字串時，之後就都會以字串方式累加。

reduce() 陣列累加 函式

陣列累加還可以用 reduce() 的方法

Array reduce() 的作法是將陣列由左至右累加之後回傳一個單一的值。

reduce() 函式

語法使用時還有陣列的索引 currentIndex 和 呼叫陣列 array 可作使用

實際用全部數字的陣列累計試試

會得到數字 7 ，和用 For 迴圈得到的值是相同的

接著再用全部字串和混用的陣列試試

會發現得到的值都會是 “1-125” ，這是因為 accumulator 在初始時會先取得初始值，若是沒有值乎略索引 0 並從索引 1 開始。

可以在其中加上 console.log(currentIndex); 觀察也可以發現索引是從 1 開始。

accumulator 的初始值也就是 For 迴圈例子中 let sum = 0; 的意思，在 reduce() 就是 initialValue ，預設 initialValue = 0 是可以省略不寫。

設定 accumulator 的初始值是要放在函式後

就會得到 “array_mix 1-125”
