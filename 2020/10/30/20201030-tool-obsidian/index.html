Obsidian 筆記整理

十月 30, 2020

關於 Obsidian

官網 https://obsidian.md/

  • 筆記工具 Obsidian

  • 免費單機軟體、付費連線功能

    • 免費只提供個人單機，所以也不需註冊就可使用
      • 付費並註冊帳號，付費服務有可以加入開發的社群、提早獲得內部的訊息、作商業用途和多台電腦的資料同步、文件上傳發布的功能。

  • 編寫格式

    • Markdown 格式保存

  • 雙向連結功能

    • 這功能會建立關鍵字卡片，並且產生和這關鍵字卡片關連的其他索引內容，更方便快速瀏覽

  • 可用外掛擴充功能，例如錄音功能、簡報功能等

  • 筆記應用的方法

    • Zettelkasten 卡片盒筆記法

安裝使用

先到官網 https://obsidian.md/ 下載

點 2 下安裝，因為沒有需要勾選的項目，所以很快就安裝好了，接著開啟 Obsidian

看不懂沒關係，可以到最下方的語言選擇切換成繁體中文

就可以看到選單界面是繁體中文

當你已經有儲存庫的資料夾時，就只要選 開啟資料夾為儲存庫 連接就會看到全部的資料。

如果沒有就 建立新的儲存庫 就可以開始使用了。

左邊的工具列

  • 收起 - 把資料夾的目錄縮小
  • 開啟快速切換 - 曾開啟過的文件紀錄
  • 查看關聯圖
  • 開啟 Markdown 格式轉換器 - 在變更格式時，會同時修改資料夾內的全部檔案格式
  • 開啟其他儲存庫 - 選擇其他資料夾開啟
  • 說明
  • 設定 - 可設定個人使用的習慣，例如
    • 限制每行的字數的 縮減行寬
    • 文字過長的 文字自動換行
    • 刪除檔案的移至資源回收筒或永久刪除的設定
    • 附件資料夾路徑的設定
    • 外觀主題的設定
    • 自訂快捷鍵設定 … 等

撰寫

因為是記錄成 md 檔，所以寫法和相同的，除了 Obsidian 的雙向連結功能

在文章內呈現的樣子

要跳到 關鍵字連結 的檔案時，按 Ctrl + 滑鼠左鍵就可開啟

在文章內加上聲音檔、PDF

折疊標題

折疊標題設定

到 設定 Options 內的 編輯器 Editor 找到 摺疊標題 Fold heading 開啟功能

就可看到在標題的左邊出現倒三角形

點一下三角形就可以收合標題下的內容，像下圖三角形變成向右並在後方加上三個小點，這用來表示這標題下還有內容

再點一下向右的三角形就可以展開內容。

當內容多的時侯，這功能就可以只把需要的部分展開就好，對編輯內容還滿方便的。

若是要折疊或展開全部內容時，當標題很多的時侯一個一個點還滿費時的，所以還可以用快捷鍵一次開合全部

快捷鍵的設定

在 設定 Options 內的 快捷鍵 Hotkeys 在上方的搜尋找出標題，就會找到 摺疊所有標題和列表 和 展開所有標題和列表 的快捷鍵設定

預設是沒設定所以可以自已設一個方便使用的

標籤面板

在文章內還可以像是貼便利貼一樣，在要 Tag 的地方加上，Tag 的數量都會出現在 標籤面板

點擊要尋找的 Tag 時就會快速把資料夾內的標籤列出來，並可以快速到連接到相對的位置

這功能十分的方便，預設是關閉的

標籤面板的設定

設定 Options > 核心外掛 Plugins > 標籤面板 Tag pane

建立標籤 #Tag
要注意 ＃ 和標籤名稱之間不能有空格，不然就會變成標題了

要開啟 標籤面板 的話，在程式的最右側點擊 ＜ 打開面板

就會看到多一個 ＃ 的面板，裡面就是全部的標籤了

反向連結

在標籤面板前的是 反向連結 面板，預設是開啟的，若是要關閉面板的話，只要關閉反向連結功能就行了。

反向連結的設定

設定 Options > 核心外掛 Plugins > 反向連結 Backlinks

這是 Obsidian 主要推薦的功能，也是在整理檔案時一個十分強大的功能。

加入最愛

最愛標記的設定

設定 Options > 核心外掛 Plugins > 最愛標記 Starred

開啟 最愛標記 後，就可在資料夾面板上方的頁籤看到星星圖案 ✩ 的最愛標記面板，

在下方的 2 個按鍵

  • 把目前的檔案加到最愛
  • 把搜尋的標籤加到最愛

在加入或刪除時都還會看到提醒

若是覺得需要打開點擊加入最愛的過程太麻煩也可以用快捷鍵

快捷鍵的設定

設定 Options > 快捷鍵 Hotkeys > 搜尋找出 最愛 star，就會找到 取消/最愛標記當前檔案 的設定

預設沒設定，所以就可以設定方便使用的組合

頁面預覽

這是可以在沒打開檔案的情況，利用反向連結的連結文字預覽內容

要顯示預覽內容只要把滑鼠移到 連結文字 上，按下 Ctrl 鍵就會像下圖出現，要打開檔案的話就按 Ctrl 鍵 + 滑鼠按鍵就行了

標籤面板的設定

設定 Options > 核心外掛 Plugins > 頁面預覽 Page preview

其他功能

在 設定 Options 內還有其他功能，像是

  • 在右側列出檔案內的標題目錄
    • 核心外掛 Plugins > 大綱 Outline
    • 開啟後就會在右側出現面板
  • 載入其他 Markdown 文件並轉換成 Obsidian 的格式
    • 核心外掛 Plugins > Markdown 格式轉換器 Markdown format importer
    • 開啟後就會在左側出現 ，就會開啟轉換工具
  • 面板的位置可以從右側拖拉到左側，相同的左側面板也可以移到右側，或是要上下排版的方式也可以喔。

因為目前只會用到這些功能，日後有用到其他功能時再來補充~
