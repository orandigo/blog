Animate.css 動畫插件

五月 24, 2020 CSS, 插件

簡介

為網頁增加動畫效果的 CSS 插件，只需要載入 CDN 並在要有動畫效果的區塊加上 Class 名稱就能看到效果了。

效果有彈跳、淡入淡出、滑入滑出、旋轉過場、放大縮小過場 … 等，可以到 Animate.css 內試用看看每個有趣的動畫。

應用的注意事項

要注意動畫效果有可能會 阻礙到使用者在觀看或使用體驗

所以官方文件也有提供動畫 最佳的做法 和 避免踩到的陷阱 的註解

  • 在網頁中呈現動畫可以提升使用者的體驗，但有幾點必須注意
    • 動畫目的在提示、轉換情境、進入、跳出 … 等讓使用者觀感更佳
    • 避免設定在 <html> 、<body> 的標籤上，因為整個頁面跳動不一定比較好還可能產生奇怪的錯誤，建議是用一個區塊包著動態的物件會比較好
    • 避免使用無止境的動畫
    • 避免修改覆寫 CSS 屬性設定，因為可能會導至動畫效果失效產生奇怪的錯誤。

簡易的修改設定

若是要微調動畫 持續時間 、 延遲時間 還可以使用簡化寫法

  • 延遲執行
    • 直接在 class 增加 animate__delay- 加上秒數，預設 1 至 5 秒
  • 控制動畫執行的速度
  • 重複執行次數
class 名稱 預設重複次數
animate__repeat-1 1
animate__repeat-2 2
animate__repeat-3 3
