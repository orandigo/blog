Victor.js 繪製 2D 向量圖

五月 17, 2020 JavaScript, library

簡介

這是繪製 2D 向量圖的套件，可以先畫好再放到 canvas 內作圖檔使用。
因為只能繪製靜態的圖，所以檔案容量很小。

官網
http://victorjs.org/

載入方式

  • bower

  • npm

  • 下載檔案

    • https://github.com/maxkueng/victor/tree/master/build
  • CDN

建立

這一行就代表建立一個有 (X, Y) 值的點，建立時不限一個點，也可以用陣列一次建立多個點
