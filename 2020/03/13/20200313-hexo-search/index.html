Hexo 站內搜尋

三月 13, 2020

搜尋工具

套用的 Theme 網頁內有搜尋功能，但只套用 Theme 還沒安裝相關的插件之前，還不能使用

在 _config.yml 找到可用的設定

  • insight : 需要再安裝 hexo-generator-json-content 才能使用

  • swiftype : 站內搜尋

    • 需要注冊帳號才能使用 Swiftype

  • Hexo 增加搜尋工具

    • hexo-generator-search 產生 xml 檔案，可用在 atom.xml
    • hexo-generator-json-content 產生 json 檔案
      • 安裝 : 當執行 hexo s 或 hexo g 時會一起自動執行
    • 預設會產生的 json 檔
    • 因為預設產生的資料多但不一定會用到，所以可以選擇要產生的搜尋資料，並降低產生出來的資料大小
      • 到 _config.yml 設定
